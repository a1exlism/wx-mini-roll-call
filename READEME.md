# Techs
* ProcessOn: 脑图设计
* 墨刀: 小程序界面原型设计
* showDoc: 在线API设计
* 官方IDE+白鹭Egret Wing: 前端代码编写
* weui-wxss: 官方UI库
* Postman: rest 接口测试/设计
* wafer: 集成部署
* LeanCloud: 云后台, 等写到后端时进行考虑
* node/php? 后端代码编写
